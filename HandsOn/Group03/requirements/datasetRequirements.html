Requisitos del Dataset

Descripción

Aquí se detallarán los requisitos que cumple el Dataset elegido "Inventario de las cámaras de seguridad de la red de infraestructuras de la ciudad de Barcelona" y por qué cumple o no estos requisitos.

Este Dataset incluye códigos y nombres de distritos y barrios de Barcelona que podrían servir de enlace con otros datasets.

Podemos encontrar otros datasets como el de Transportes de la ciudad de Barcelona que incluyen también códigos y nombres de barrios y distritos de la ciudad de Barcelona por lo que podría ser enlazado con el dataset de cámaras de seguridad de Barcelona.

Requisitos de la Aplicación y Justificación

  • R1:Data are in one of the smart city domains.

    Justificación: Cumple con el requisito porque las cámaras de seguridad están directamente relacionadas con el dominio de la seguridad pública, que es una de las áreas clave en las ciudades inteligentes.

  • R2: Data are available as a CSV file.

    Justificación: Cumple con el requisito porque el dataset de las cámaras de seguridad está disponible en formato CSV.

  • R3: Data have an open license so they can be published.

    Justificación: El dataset cuenta con una licencia abierta. Concretamente estos son los permisos:

    Share — copy and redistribute the material in any medium or format for any purpose, even commercially.

    Adapt — remix, transform, and build upon the material for any purpose, even commercially.

    Licencia

  • R4:Data can be easily linked with generic real-world entities (e.g., locations).

    Justificación: Cumple con el requisito porque en el dataset cada cámara de seguridad está geolocalizada, lo que permite vincular las ubicaciones de las cámaras con entidades del mundo real, ya que en el dataset se indica en qué distrito, barrio y coordenadas se encuentran.

  • R5: Documentation exists for the data (optional)

    Justificación: El Dataset no incluye documentación por lo que no se cumple este requisito.

  • R6: Data can come from multiple data sources (optional).

    Justificación:La fuente de los datos de este Dataset es únicamente el Ayuntamiento de Barcelona por lo que no se cumple este requisito.
