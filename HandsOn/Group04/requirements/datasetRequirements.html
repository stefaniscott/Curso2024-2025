R1: Data in smart city domain

The dataset comes from the datos.gov.uk website, which collects data from United Kindom.

R2: Data available as a CSV file

The dataset is available as a CSV file here, and can be located locally at the csv folder of the repository.

R3: Data have an open license so it can be published

The data can be reused for commercial and non-commercial purpose under an open license.

R4: Data can be easily linked with generic real-world entities

The dataset on abandoned vehicles can be easily linked to the real world, as it provides concrete information about an issue that occurs in everyday life. Can be connected to specific locations within the UK, allowing for the identification of geographic patterns in vehicle abandonment across different regions or cities. This type of dataset has direct implications for criminological studies, as abandoned vehicles can be linked to illegal activities such as theft, vandalism, or even more serious crimes.
