Application Requirements

Application Overview

This application will provide an interface to monitor and analyze emergency medical interventions (SAMUR) across Madrid. It will allow city officials and emergency coordinators to visualize intervention data in real-time, identify trends, and optimize response strategies.

Key Functional Requirements

  • Data Visualization: Display updates of SAMUR interventions in a dashboard overview
  • Historical Data Analysis: Provide tools to analyze historical intervention data to identify patterns over time.
  • Location-Based Filters: Enable users to filter interventions by specific districts, or hospitals within Madrid.
  • Emergency Type Filters: Filter interventions by the type of emergency (e.g., traffic accidents, medical emergencies).

User Roles and Permissions

  • City Officials: Ability to import and update the datasets for each year
  • Public Users: Access to all data and insights, as well as the ability to export reports
