Dataset Requirements

R1: Data in smart city domain

The dataset comes from the datos.got.es website, which collects data from Madrid, Spain.

R2: Data available as a CSV file

The dataset is available as a CSV file here, and can be located locally at the csv folder of the repository.

R3: Data have an open license so it can be published

The data can be reused for commercial and non-commercial purpose under an open license.

R4: Data can be easily linked with generic real-world entities

The dataset contains details about real-world events (traffic accidents) that are directly linkable to physical entities like streets, intersections, vehicles and people of the city Madrid. For instance, by including precise street names, the data can be easily be linked with geographical information systems.
