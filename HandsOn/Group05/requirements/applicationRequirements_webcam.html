Application Requirements

Overview

The application will be built on top of the webcam dataset and will provide real-time access to webcam feeds for monitoring traffic, weather conditions, and tourism hotspots.

Key Requirements

  • The application should provide an interactive map showing webcam locations.
  • Users should be able to click on locations to view live or recorded footage.
  • There should be filters for different regions or categories (e.g., traffic, weather).
  • Integration with weather and traffic APIs for context.

Mock-ups

The application will feature:

  • A homepage with an interactive map interface.
  • Each webcam marker will open a window with video and additional data.
